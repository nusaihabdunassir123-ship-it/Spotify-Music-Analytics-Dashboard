Spotify Music Analytics Dashboard 🎵
📌 Project Overview

The Spotify Music Analytics Dashboard is an interactive Power BI project designed to analyze Spotify music data and provide meaningful insights into song trends, user preferences, and audio features.

This dashboard helps users understand music listening behavior through KPIs, charts, filters, and visual analytics.

🎯 Project Objective
Analyze Spotify song data using Power BI
Visualize music trends and audio features
Compare liked vs non-liked songs
Identify patterns in danceability, energy, and tempo
Create an interactive and user-friendly dashboard
📊 Dashboard Features
🎵 Average Danceability Analysis
⚡ Average Energy Tracking
🥁 Tempo Insights
❤️ Liked Songs Count
📈 Energy Trend by Tempo
📉 Danceability vs Energy Scatter Analysis
🍩 Song Preference Distribution
🎚️ Interactive Filters & Slicers
